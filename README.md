## Circuits Built

### 1. Half Adder
- **Inputs:** A, B
- **Outputs:** Sum, Carry Out
- **Logic:** Sum = A XOR B | Carry = A AND B
- **Tool:** Tinkercad Circuits
- **Status:** ✅ Complete and verified

### 2. Full Adder
- **Inputs:** A, B, Carry In
- **Outputs:** Sum, Carry Out
- **Logic:** Built from two Half Adders + OR gate
- **Tool:** Tinkercad Circuits
- **Status:** ✅ Complete and verified
